# Data Visualization Course - README

## Course Overview
This is a comprehensive Data Visualization course covering multiple libraries and techniques for visualizing data in Python.

## Labs Overview

| Lab | Topic | Key Focus |
|-----|-------|-----------|
| **Lab 1** | Introduction to Pandas & Histograms | Data loading, exploration, and histogram visualization |
| **Lab 2** | Bar Plots | Bar chart creation using Matplotlib, Seaborn, Plotly, Altair |
| **Lab 3** | Athlete Sport Event | Data analysis on sports/athlete datasets |
| **Lab 4** | Scatter Plots, Line Plots & Heatmaps | Advanced plot types using MPG dataset |
| **Lab 5** | Temporal Data Visualization | Time series analysis and visualization |
| **Lab 6** | Bokeh for Temporal Data | Interactive visualizations with Bokeh library |
| **Lab 7** | Evaluation - Time Series with Bokeh | Exam covering Bokeh time series visualization |
| **Lab 8** | Text Data Visualization | Visualizing and analyzing text datasets |
| **Lab 9** | Text Analysis | NLP and text processing techniques |
| **Lab 10** | Evaluation - Text Analysis | Sentiment analysis and text visualization |

## Main Libraries Covered
- **Pandas**: Data manipulation and exploration
- **Matplotlib**: Static plot generation
- **Seaborn**: Statistical visualization
- **Plotly**: Interactive visualizations
- **Altair**: Declarative visualization
- **Bokeh**: Interactive web-based plots
- **NLTK & TextBlob**: NLP and text analysis

## Datasets Used
- Diamonds dataset
- MPG (Miles Per Gallon) dataset
- Melbourne Daily Minimum Temperatures
- Amazon Reviews dataset
- Jupyter notebooks folder with supporting materials
